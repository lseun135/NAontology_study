1. 초기 계획

  • 대상: 외부 이용자
  • 범위: 제 21대 국회 회의록 (본회의, 상임위원회)
    • 379회(2020.06.05) ~ 414회(2024.05.29)
  • RiC-O 사용 + 외부 온톨로지 부분적 재사용 → 직접 재사용

1.1 역량 질의(CQs)

항목 Competency Questions (CQ) 예시
회의록 중심 검색
  • 특정 날짜 또는 회기에서 열린 회의록을 찾을 수 있는가?
  • 특정 기간 동안 개최된 모든 회의록을 일괄 조회할 수 있는가?
  • 특정 키워드(주제어)가 언급된 회의록을 찾을 수 있는가?
  • 특정 법령 조항이 인용되거나 참조된 회의록을 찾을 수 있는가?
의원 중심 탐색
  • 특정 국회의원의 발언이 포함된 회의록을 찾을 수 있는가?
  • 특정 국회의원의 직책에 따른 발언을 구분하여 찾을 수 있는가?
  • 특정 의원이 발의한 안건 목록을 확인할 수 있는가?
법률안/안건 중심 탐색
  • 특정 법률안/안건이 논의된 회의록을 찾을 수 있는가?
  • 특정 법률안/안건이 어느 상임위원회에서 심의되었는지 확인할 수 있는가?
  • 특정 법률안의 심의 경과를 추적할 수 있는가?
  • 본회의 및 상임위 회의록을 연결하여 찾을 수 있는가?
  • 안건의 심의 진행 단계를 확인할 수 있는가?
회의 유형 및 절차 탐색
  • 본회의 또는 상임위원회 등 회의 종류에 따라 회의록을 구분하여 조회할 수 있는가?
  • 특정 상임위원회 소속 회의록을 찾을 수 있는가?

Record

m1-record

메인이 되는 모듈. RiC 온톨로지의 구조를 사용하되, 국회 보존기록물의 기본적인 세 가지 분류체계를 스키마로 추가

참고: 국회도서관 국회기록보존소. (2021). 국회 보존기록물 정리·기술 지침

Person

m1-person

인물, 조직 표현. Popolo project의 Person, Organization, Membership, Area 클래스 참고

조직 분류

  • 국회 내부 조직 분류를 위해 nam2:OrgType 클래스 생성
  • 하위 클래스: Institution, Party, Committee
  • Committee 하위에 StandingCommittee, SpecialCommittee

국회법 제35조: 국회의 위원회는 상임위원회와 특별위원회 두 종류로 한다.
