1.1 역량 질의(CQs)

항목 Competency Questions (CQ) 예시
회의록 중심 검색
  • 특정 날짜 또는 회기에서 열린 회의록을 찾을 수 있는가?
  • 특정 기간 동안 개최된 모든 회의록을 일괄 조회할 수 있는가?
  • 특정 키워드(주제어)가 언급된 회의록을 찾을 수 있는가?
  • 특정 법령 조항이 인용되거나 참조된 회의록을 찾을 수 있는가?
의원 중심 탐색
  • 특정 국회의원의 발언이 포함된 회의록을 찾을 수 있는가?
  • 특정 국회의원의 직책에 따른 발언을 구분하여 찾을 수 있는가?
  • 특정 의원이 발의한 안건 목록을 확인할 수 있는가?
법률안/안건 중심 탐색
  • 특정 법률안/안건이 논의된 회의록을 찾을 수 있는가?
  • 특정 법률안/안건이 어느 상임위원회에서 심의되었는지 확인할 수 있는가?
  • 특정 법률안의 심의 경과를 추적할 수 있는가?
  • 하나의 법안에 대해 본회의 및 상임위 회의록을 연결하여 찾을 수 있는가?
  • 안건의 심의 진행 단계를 확인할 수 있는가?
회의 유형 및 절차 탐색
  • 본회의 또는 상임위원회 등 회의 종류에 따라 회의록을 구분하여 조회할 수 있는가?
  • 특정 상임위원회 소속 회의록을 찾을 수 있는가?
